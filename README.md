eHealth Glossary App
====================

> simple app for the eHealth glossary app

This app is meant as a usefull angular learning project.

## Setup

```
git clone git@github.com:zoepage/ehealth-glossary-app.git
cd ehealth-glossary-app
npm install
bower install
grunt serve
```
